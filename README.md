## Описание проекта

Stream Vibe — это фронтенд многостраничного приложения стримингового сервиса, который состоит из 6 полноценных страниц со множеством интерактивных элементов.

## Функционал:

- Меню оверлей и бургер кнопка;
- Переключение между табами;
- Кастомные видеоплееры;
- Аккордеоны с вопросами и ответами;
- Маска для поля ввода телефона;
- Кастомный компонент Select;
- Слайдеры для фильмов и контента;
- Адаптивный интерфейс;
- Страницы: Главная, Фильмы, Подписки, Поддержка, Детали фильма/шоу.

## Стек технологий:

- HTML5;
- CSS3;
- Препроцессоры: SASS/SCSS;
- Методология БЭМ;
- Верстка по макету Figma;
- JavaScript:
  - Стиль CamelCase;
  - Парадигма ООП;
  - Промисы;
  - Функции, преобразующие все пиксели (px) на сайте в адаптивные единицы измерения rem;
- JSX;
- Minista — фреймворк для статических сайтов;
- Vite — инструмент сборки;
- PostCSS с автоматическим преобразованием px в rem.

## Установка и запуск приложения:

Клонировать репозиторий:

    git clone https://github.com/i-matsnev/stream-vibe.git

Установить зависимости:

    npm install

Запустить dev-сервер:

    npm start

Собрать проект для production:

    npm run build

Предпросмотр production-сборки:

    npm run preview
